## Context

В раскладке `winkeys-pd` (`ru`) уже есть несколько F-шорткатов для символов, которых нет на русской раскладке в базовом доступе: `:`, `~`, `"`, `'`, `?`, `|`, `№`. Символ `[` (bracketleft) отсутствует — его можно ввести только переключившись на латинскую раскладку.

## Goals / Non-Goals

**Goals:**
- Добавить `[` на Shift+F1 в раскладке `winkeys-pd`
- Сохранить F1 без Shift как обычный F1

**Non-Goals:**
- Не добавлять `]` — для него пока нет потребности
- Не менять другие раскладки
- Не менять evdev.extras.xml

## Decisions

- **Формат**: `key <FK01> { [ F1, bracketleft ] };` — соответствует стилю остальных F-шорткатов в `winkeys-pd`
- **Место**: строка вставляется перед `key <FK02>` для логической группировки по порядку F-клавиш
- **Имя клавиши**: `bracketleft` — стандартное имя keysym в XKB

## Risks / Trade-offs

- Нет рисков — изменение изолированное, одна строка, обратно совместимое
