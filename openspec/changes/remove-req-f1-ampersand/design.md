## Context

spec req-12 определяет Shift+F1 → `&`. При этом `&` уже доступен на Shift+AE07 (req-08) в базовом определении `winkeys-pd`. Удаление spec req-12 и соответствующей строки из патча не затрагивает никакие другие требования.

## Goals / Non-Goals

**Goals:**
- Удалить spec req-12-shift-f1-ampersand
- Удалить FK01-binding из `winkeys-pd` в патч-файле
- Проверить, что обновлённый патч накатывается утилитой `patch` корректно

**Non-Goals:**
- Любые другие изменения раскладки
- Использование `git` или `patch` для создания патча

## Decisions

1. **Простое удаление.** FK01 не используется ни в каких других spec, удаление безопасно.
2. **spec удаляется полностью** — вся директория req-12.
3. **Патч-файл редактируется напрямую** — удаляется строка 29 с FK01, правится комментарий о списке изменений (убрать `Shift+F1 &`), корректируется счётчик строк в hunk-заголовке.
4. **Верификация:** копируем `ru_orig` → `ru_test`, накатываем обновлённый патч через `patch`, сравниваем `ru_test` с эталонным `ru` (полученным тем же удалением FK01 из рабочего `ru`). Если `diff` пустой — патч корректен.

## Risks / Trade-offs

- Ошибка в ручном подсчёте строк hunk-заголовка — верификация `diff`'ом отлавливает
